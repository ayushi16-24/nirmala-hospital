# Nirmala Women's Care - Django Project

This is a fully portable Django project for Nirmala Women's Care Hospital. All images and assets are contained within the project folder.

## Setup Instructions

### Step 1: Extract ZIP
Extract the project folder to your desired location.

### Step 2: Open Terminal
Open your terminal (Command Prompt, PowerShell, or Terminal) and navigate to the extracted project folder.

### Step 3: Install Dependencies
Install the required Python packages using the following command:
```bash
pip install -r requirements.txt
```

### Step 4: Run Server
Start the development server:
```bash
python manage.py runserver
```

### Step 5: Access the Website
Open your browser and go to:
`http://127.0.0.1:8000/`

---

## Project Structure
- `nirmala_hospital/`: Project configuration (settings, urls).
- `core/`: Main application logic (views, models, templates).
- `static/`: Static assets (CSS, JS, Images).
- `media/`: User-uploaded media files.
- `manage.py`: Django management script.
- `db.sqlite3`: Local database file.
- `requirements.txt`: List of dependencies.
